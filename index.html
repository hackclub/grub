Requirements

  • You are a high school student
  • Your website is using only HTML and Tailwind CSS. Pure CSS may only be used where something can't be done without it, such as custom animations.
  • Make it responsive! It should be useable on both phones and computers!
  • The website must use containers!
  • The website must include content and have meaning!
  • Your website should contain less than 15% of Javascript
  • Have at least 2 unique features. It can include, but not limited to:
    • Navbar
    • Footer
    • Sidebar
    • Any larger components like blog cards, FAQ sections, or other interactive elements. (Responsive design is not considered as a unique feature, it is a requirement)

Frequently Asked Questions

What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework that provides low-level utility classes to build custom designs. It is a CSS framework that allows you to build your website without writing any CSS. You can find more about it here.

Can I use CSS in my project?

Depends. Tailwind CSS allows you develop your website using only HTML and Tailwind CSS in most cases. Your website should only be using HTML and Tailwind CSS. Pure CSS may only be used where something can't be done without it, such as custom animations.
